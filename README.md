# PORTAFOLIO DE PROYECTOS

Portafolio de proyectos creado para la materia Proyecto Integrador.
Este repositorio presenta una selección de proyectos desarrollados con el objetivo de demostrar habilidades y conocimientos adquiridos durante la formación académica en el área de desarrollo de software.

---

## Proyecto 1: EduGlobal - Plataforma Educativa

**Objetivo:**
Colaborar en el desarrollo de un sitio web informativo para el centro educativo "EduGlobal", permitiendo la visualización de oferta académica y registro de alumnos.

**Tecnologías utilizadas:**
HTML5, CSS3, Bootstrap 5, Git/GitHub.

**Descripción:**
Proyecto colaborativo de un sitio web multipágina (Inicio, Cursos, Registro). La plataforma utiliza Bootstrap para asegurar un diseño responsivo y uniforme entre las diferentes secciones desarrolladas por los miembros del equipo.

**Rol en el proyecto:**
Trabajo en equipo. Mi responsabilidad fue el **desarrollo completo de la vista de "Cursos"**.

* **Maquetación:** Estructura HTML semántica para el catálogo de clases.
* **Componentes:** Implementación de Cards de Bootstrap para los cursos y tablas estilizadas para los horarios.
* **Integración:** Unificación de la barra de navegación y el footer acordados por el equipo para mantener la consistencia visual.

**Aprendizajes obtenidos:**
* Colaboración y control de versiones con Git.
* Diseño modular utilizando componentes de Bootstrap.
* Estandarización de estilos en proyectos grupales.

## Proyecto 2: SUPI – Sistema Universitario Privado Integral

**Objetivo:**
Desarrollar una plataforma web privada que permita a los estudiantes consultar y localizar información institucional de los docentes de la Universidad Politécnica de Quintana Roo.

**Tecnologías utilizadas:**
PHP, MySQL, XAMPP, Visual Studio Code.

**Descripción:**
Sistema web con autenticación que centraliza la información académica. La aplicación gestiona dinámicamente los datos de los profesores, materias y horarios a través de una base de datos relacional, permitiendo consultas rápidas y seguras.

**Rol en el proyecto:**
Trabajo en equipo. Mi participación fue técnica, enfocándome en el **diseño y estructuración de la Base de Datos**.

* **Modelado de datos:** Definición de las entidades, atributos y relaciones necesarias para el sistema (Docentes, Horarios, Usuarios).
* **Creación de estructura:** Desarrollo de las tablas en MySQL y establecimiento de claves foráneas para asegurar la integridad referencial.
* **Optimización:** Organización de la información para evitar duplicidad y mejorar la eficiencia de las consultas.

**Aprendizajes obtenidos:**
* Se reforzaron habilidades en lenguaje SQL y uso de herramientas de gestión como phpMyAdmin.
* Comprensión profunda de la normalización de bases de datos y la lógica relacional.
* Integración de base de datos con lenguajes de servidor (PHP).

## Proyecto 3: Panel de Inventario para Farmacia

**Objetivo:**
Desarrollar una aplicación web para la gestión de inventario de medicamentos, permitiendo el registro, consulta, edición y eliminación de información de manera segura y organizada.

**Tecnologías utilizadas:**
PHP, MySQL, HTML5, CSS3, Bootstrap 5, JavaScript.

**Descripción:**
Aplicación web dinámica que gestiona el flujo de datos de una farmacia. El sistema permite a los administradores mantener un control exacto del stock mediante operaciones de base de datos en tiempo real, garantizando la integridad de la información y el acceso restringido mediante autenticación.

**Rol en el proyecto:**
Trabajo en equipo. Mi responsabilidad principal fue el **desarrollo del Backend**.

* **Lógica del servidor:** Programación de las funciones en PHP para procesar las solicitudes del usuario.
* **Implementación CRUD:** Desarrollo de las sentencias SQL y código PHP para Crear, Leer, Actualizar y Eliminar registros de medicamentos y proveedores.
* **Gestión de Sesiones:** Implementación del sistema de login y seguridad para proteger las rutas administrativas, asegurando que solo usuarios autenticados accedan al panel.

**Funcionalidades principales (Backend):**
* **Sistema de Autenticación:** Validación de credenciales y manejo de variables de sesión (`$_SESSION`).
* **Gestión de Inventario:** Lógica para insertar nuevos productos y actualizar existencias.
* **Consultas Dinámicas:** Filtrado de productos por categorías mediante consultas SQL parametrizadas.
* **Conexión a Base de Datos:** Integración eficiente entre la interfaz web y MySQL.

**Aprendizajes obtenidos:**
* Dominio de la interacción entre PHP y MySQL.
* Comprensión del ciclo de vida de una petición web (Request/Response) y manejo de datos de formularios.
* Lógica de programación para la manipulación y validación de datos del lado del servidor.
